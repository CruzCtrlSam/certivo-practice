# Certivo Practice Web App

Static practice-test web app for InsTX-Life21 Texas Life Agent practice.

## What is included
- 300 questions from the six simulators
- Practice mode with instant feedback
- Exam mode with scoring at the end
- Randomized questions and answer choices
- Missed-question review
- Saved progress using the browser's localStorage
- Dashboard by simulator

## Fastest way to publish on Vercel
1. Go to https://vercel.com/new
2. Create a free account or sign in.
3. Choose **Import** / **Deploy manually**.
4. Drag this entire folder, or drag the ZIP file after unzipping.
5. Vercel will publish a live URL.

## Fastest way to publish on Netlify
1. Go to https://app.netlify.com/drop
2. Drag this entire folder into the upload box.
3. Netlify will publish a live URL.

## iPhone use
Open the live URL in Safari, then tap Share > Add to Home Screen.
